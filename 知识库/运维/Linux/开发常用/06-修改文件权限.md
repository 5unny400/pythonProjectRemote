# 修改文件权限

```shell
# 当前目录下所有文件赋予权限(读、写、执行) 给 所有用户
chmod -R 777 ./xx
```

权限说明

```shell
-rw-r--r-- 1 root root     4 1月   3 17:37 b.txt
-rw-r--r-- 1 root root 87765 1月   3 17:25 README.md
-rwxrwxrwx 1 root root 74552 1月   3 17:20 zhengqingya.sh
```

| 参数  | 含义                          |
|-----|-----------------------------|
| -r  | 赋予读取权限                      |
| -w  | 赋予写入权限                      |
| -x  | 赋予执行权限                      |
| 777 | 赋予可读、可写、可执行权限（读：4，写：2，执行：1） |
