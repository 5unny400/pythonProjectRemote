### 统计文件或文件夹的大小

```shell
# 查看占用空间比较大的文件夹并按大小排序输出
du -s /* | sort -nr
du -s -h /var/* | sort -nr

# 查找home目录100M以上的文件
find /home/ -size +100M


# 查看磁盘大小，及分区情况
df -h

# 分别查询当前目录下的各目录的总空间大小
du -hd1

# 查看back目录的大小
du -sh back

# 查看当前目录的各文件大小
du -sh *

# 查看大于G的文件
du -sh * | grep G

# 查看大于M的文件
du -sh * | grep M
```
