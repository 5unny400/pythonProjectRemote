# 插槽

### 父组件

```
<template>
  <HelloWorld>
    <template v-slot:left>具名插槽-left</template>
    <template #right>具名插槽-right</template>
  </HelloWorld>
</template>
<script setup>
import HelloWorld from "./components/HelloWorld.vue";
</script>
<style scoped></style>
```

### 子组件

```
<template>
  <h3>hello</h3>
  <p>
    <slot name="left"></slot>
  </p>
  <p>
    <slot name="right"></slot>
  </p>
</template>
<script setup></script>
<style scoped></style>
```
