# CSS选择器

## 一、基础选择器

### 1、标签选择器

用html标签名称作为选择器，按标签名称分类，为页面中某一类标签指定统一的css样式。

```
/* 为所有的<div>标签统一设置样式 */
div {
    color: red;
}
```

### 2、类选择器

```
<div class="div-red">hello</div>
```

```
.div-red {
    color: red;
}
```

多类名中间用空格分隔

```
<div class="div-red div-font">hello</div>


.div-red {
    color: red;
}
.div-font {
    font-size: 20px;
}
```
