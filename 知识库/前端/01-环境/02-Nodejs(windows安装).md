### Node.js

> tips: 建议通过nvm安装，方便进行版本管理。

[点击下载](https://nodejs.org/zh-cn/download/)

```shell
# 查看版本
node -v
npm -v

# npm 升级
npm install -g npm
# 指定版本
# npm -g install npm@8.5.4

# node升级
# 安装n模块
npm install -g n
# 升级到最新稳定版
n stable
```

#### 卸载

```
# 控制面板卸载 node.js
# 删除`C:\Users\Administrator\AppData\Roaming`目录下的 `npm`和`npm-cache` 文件夹
# 删除环境变量
```
